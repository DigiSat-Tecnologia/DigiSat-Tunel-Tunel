# bunl

## A Bun WebSocket re-write of LocalTunnel

To try it:

```bash
bun x bunl -p 3000 -d dev.rubric.me -s my-name
```

To install dependencies:

```bash
bun i
```

To run the client:

```bash
bun lt
```

To run the server:

```bash
bun server
```

To run a dummy server on localhost:3000:

```bash
bun demo
```
